# 미션 요구사항 체크리스트

## 산출물

- [x] AI API 연동 및 자동화 흐름이 단일 실행으로 끝까지 동작한다
- [x] 변경 요약·커밋 메시지·PR 초안이 터미널에 출력된다
- [x] 프로젝트가 GitHub 원격 저장소에 push 되어 있다
- [x] 커밋 히스토리를 확인할 수 있다
- [x] 브랜치 작업 흐름을 확인할 수 있다
- [x] README.md만 보고 도구를 실행할 수 있다

## 설명 요구사항

- [x] AI API를 REST 방식으로 연동하는 요청 구성 → 응답 처리 → 예외 대응 흐름
- [x] `temperature`, `max_tokens` 등 파라미터가 결과 품질에 주는 영향
- [x] `git status`, `git diff` 결과를 프로그램 입력으로 연결하는 방식과 자동화 흐름
- [x] Commit/PR 양식과 코드 변경 맥락을 포함한 프롬프트 구성 원리
- [x] 생성 텍스트를 실무 규칙에 맞게 검증·보정하는 이유와 방법

## 기능 요구사항

### Git 변경 사항 수집

- [x] Git이 초기화된 프로젝트 루트에서 실행된다
- [x] `git status` 결과를 수집해 변경 파일 목록을 확인할 수 있다
- [x] `git diff` 결과를 수집해 diff 텍스트를 얻을 수 있다
- [x] 변경 사항이 없으면 안내 메시지를 출력하고 종료한다

### AI API 연동

- [x] API Key를 환경변수로 설정하며 코드에 하드코딩하지 않는다
- [x] CLI 실행 시 AI API가 호출되고 생성 텍스트가 터미널에 출력된다
- [x] 호출 실패 시 오류 원인을 포함한 메시지를 출력한다
- [x] `--model`을 CLI 옵션으로 변경할 수 있고 기본값이 있다
- [x] `--temperature`를 CLI 옵션으로 변경할 수 있고 기본값이 있다
- [x] `--max-tokens`를 CLI 옵션으로 변경할 수 있고 기본값이 있다

### 커밋 메시지 자동 생성

- [x] `commit` 명령으로 커밋 메시지가 생성되어 출력된다
- [x] 커밋 메시지가 변경 사항 요약을 기반으로 생성된다
- [x] 커밋 제목 1줄이 필수로 포함된다
- [x] 커밋 본문이 선택적으로 포함된다
- [x] 본문 포함 시 "변경 파일 1~3개 언급" 또는 "핵심 변경 1~2개 불릿"이 보장된다
- [x] 출력된 메시지를 복사해 그대로 적용할 수 있다

### PR 제목/본문 자동 생성

- [x] PR 제목/본문 초안을 생성하는 명령이 있다
- [x] PR 본문에 `## Why` 섹션 헤더가 포함된다
- [x] PR 본문에 `## What` 섹션 헤더가 포함된다
- [x] PR 본문에 `## How to Test` 섹션 헤더가 포함된다
- [x] 각 섹션에 최소 1개 이상의 불릿이 포함된다
- [x] PR 제목이 1줄로 출력되며 본문과 함께 확인할 수 있다

### 출력 형식 검증 및 다듬기

- [x] 커밋 제목 50자 이내 권장 / 최대 72자 규칙이 적용된다
- [x] PR 제목 최대 80자 규칙이 적용된다
- [x] PR 본문의 섹션 헤더 및 섹션별 불릿이 검증된다
- [x] 검증 후 재생성 또는 후처리 중 하나를 적용한다
- [x] 최종 출력이 구분선/헤더로 구획되어 표시된다

### 리포지토리 및 문서화

- [x] 결과물이 GitHub 리포지토리에 push 되어 있다
- [x] README에 설치 및 실행 방법이 있다
- [x] README에 환경변수(API Key) 설정 방법이 있다
- [x] README에 커밋/PR 생성 명령 사용 예시가 있다
- [x] README에 출력 예시(커밋 메시지 / PR 본문)가 있다
- [x] README에 민감정보 대응 또는 비용/요청 제한 중 1개 이상이 있다

### 개발 환경 및 제약 사항

- [x] Python 3.10 이상에서 동작한다
- [x] 웹 화면 없이 터미널에서 실행된다
- [x] 1회 실행 시 API 요청이 1~2회 이내다
- [x] 로그에 API 호출 횟수를 출력한다
- [x] Git 수집 범위가 `git status` / `git diff`로 제한된다
- [x] `git push`, GitHub PR 생성 등 원격 반영 기능이 없다
- [x] safe-mode로 마스킹 또는 diff 전송 제한 중 1개 이상을 제공한다
- [x] 생성 문구가 최종 정답이 아님을 안내한다
